# Task
Python Script to Fetch Crypto Options Data
```
Create a script capable of retrieving the full options chain for Ethereum (ETH) and Bitcoin (BTC) from the Derive.xyz platform. The script must return detailed options data.

Requirements:
- The script should provide data for each option, including: Type (Put/Call), Expiry, Strike, Bid Size, Bid Price, Ask Price, Ask Size, Volume, Implied Volatility, Delta.
- It should output structured data in a Pandas DataFrame.

Deliverables:
A Python script that fetches and returns the requested data in a structured format (Pandas DataFrame).
```
